This is a beginner-level website built with **HTML5** and **CSS3** for my _Web Programming 1_ class (Brazilian Portuguese).

## Assignment Instructions
**Interdisciplinary Project - 1st Module**
* **Deadline:** November 4th and 7th.
* **Goal:** Create a website with at least **ten (10) pages** and an external `.css` file.

**Requirements:**
* **Home Page:** Must have versions in Portuguese and English (2 pages total).
* **Contact Page:** Must include a contact form layout and a Google Maps location (Iframe) (1 page).
* **Content Pages:** Seven (7) additional pages featuring images, videos, tables, Image Maps, Iframes, and proper HTML tag formatting.
* **Styling:** All styling across the ten pages must be done through a single **external CSS file**.
* **Work Format:** Individual project.

**Allowed Themes:**
Education, Environment, Urban Mobility, Health, or Security. (Note: A theme of your own preference is also allowed).
